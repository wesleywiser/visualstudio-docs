# IDiaEnumInputAssemblyFiles::get_Count

Retrieves the number of input assembly files.

## Syntax

```c++
HRESULT get_Count ( 
   LONG* pRetVal
);
```

#### Parameters

`pRetVal`

[out] Returns the number of injected sources.

## Return Value

If successful, returns `S_OK`; otherwise, returns an error code.
